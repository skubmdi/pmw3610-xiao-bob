# PMW3610-SUDU XIAO Breakoutboard
This product is a breakout board for the PMW3610-SUDU.
<br>It features a compact design that shares the same form factor as the Seeed Studio XIAO series.
<br>It operates exclusively at 3.3V and is designed to be used in conjunction with a XIAO module.
